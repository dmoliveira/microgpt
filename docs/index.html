Context first

microgpt is a minimal, dependency-free GPT training and inference loop. It is useful as a learning artifact because every primitive is explicit: tokenization, autograd, transformer forward pass, optimizer update, and autoregressive sampling.

What this implementation teaches

How language modeling works end-to-end without framework magic or hidden abstractions.

Why this matters

Once you understand this file, larger LLM stacks become easier to reason about and debug.

Mental model

Predict next token repeatedly: train with cross-entropy, backpropagate gradients, then sample one token at a time.

Dataset bootstrapping Focus excerpt

Step-by-step explanation (after code)

1) Build a tiny corpus

Names dataset is fetched if needed, shuffled, and split into character-level docs. 

if not os.path.exists('input.txt'):
    import urllib.request
    urllib.request.urlretrieve(names_url, 'input.txt')
docs = [line.strip() for line in open('input.txt') if line.strip()]
random.shuffle(docs)

2) Turn characters into a vocabulary

Character set defines token IDs. BOS extends vocabulary for sequence starts. 

uchars = sorted(set(''.join(docs)))
BOS = len(uchars)
vocab_size = len(uchars) + 1

3) Backprop from first principles

Each operation records local derivatives, then backward() performs reverse topological traversal. 

for v in reversed(topo):
    for child, local_grad in zip(v._children, v._local_grads):
        child.grad += local_grad * v.grad

4) Run transformer + language head

Token embeddings and positional embeddings feed attention layers; logits are projected by lm_head. 

state_dict = {
    'wte': matrix(vocab_size, n_embd),
    'wpe': matrix(block_size, n_embd),
    'lm_head': matrix(vocab_size, n_embd),
}

Usage examples

Train from scratch

python microgpt.py

Runs dataset prep, training loop, and sample generation in one file.

Tweak model size

n_layer = 2
n_embd = 32
n_head = 4

Increase representational power while keeping code architecture unchanged.

Switch to your corpus

docs = [line.strip() for line in open('my_corpus.txt') if line.strip()]

Use a domain-specific dataset to generate new token distributions.
